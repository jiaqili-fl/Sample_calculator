Sample Size Calculator

Slide 2: Background

Why is Sample Size Calculation Important?

  • Ensures sufficient statistical power
  • Prevents underpowered or overpowered studies
  • Aids in cost-effective research design

Key Features

  2. Dynamic Output:

Slide 5: Overlap of two distributions

library(ggplot2)
# Create a simple normal distribution plot
var <- 1; mean1<-0;mean2<-2.3;x<-seq(-5,8,length.out=1000)
density1 <- dnorm(x, mean = mean1, sd = sqrt(var))
density2 <- dnorm(x, mean = mean2, sd = sqrt(var))
# plot
plot(x, density1, type = "l", col = "blue", lwd = 2,
    ylim = c(0, max(c(density1, density2))),
    xlab = "Value", ylab = "Density",
    main = "Two Normal Distributions")
lines(x, density2, col = "red", lwd = 2)
legend("topright", legend = c("Group 1", "Group 2"),
       col = c("blue", "red"), lwd = 2, bty = "n")
